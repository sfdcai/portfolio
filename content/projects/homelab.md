# Home Lab Infrastructure

A fully self-hosted environment used for learning enterprise infrastructure, networking, and system design.

---

## Stack

- Proxmox virtualization
- OpenWRT router setup
- pfSense firewall
- Home Assistant automation
- Netdata monitoring

---

## Capabilities

- VM orchestration
- Containerized workloads
- Network segmentation
- Observability dashboards
- Self-hosted services

---

## Purpose

This environment acts as a personal R&D lab for:
- Infrastructure engineering
- Networking experiments
- AI system testing
- DevOps workflows
